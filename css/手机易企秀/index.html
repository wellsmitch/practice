新客户礼遇

活动时间：
申请及交易达标时间：2019年1月1日至2019年2月28日
领取时间：2019年2月1日至2019年3月31日

活动对象：
申请颜卡系列的中信银行信用卡新户（附属卡用户不单独参与活动，交易计入主卡）

中信银行信用卡新户定义：名下未持有任何中信银行信用卡的客户（蓝卡或附属卡客户除外），含销卡180天以上的客户；

活动内容：

1、活动期间，申请颜卡系列的中信银行信用卡新户，于卡片批核60天内（含），累计6笔计积分交易≥99元（需每笔交易都≥99元），且其中任意一笔交易为网络指定渠道交易（微信、支付宝、美团、京东4个渠道），即可获得“颜卡新客户礼遇”权益，并凭此权益用9个积分兑换价值399元TUNE205BT JBL蓝牙耳机（曜石黑或玫瑰金）一副。“颜卡新客户礼遇”权益在活动期间兑换均有效，客户获得权益后于指定时间至中信银行“动卡空间”APP领取。礼品每日限量，先到先得，具体数量以兑换页面展示为准。

领取方式：

*客户达标5天后可登录“动卡空间APP”进行查询并兑换，活动期间每日均可尝试兑换。首批兑换时间为2019年2月1日9点整。兑换路径：动卡空间APP—我的（登录账户）—我的活动—新客户首刷礼遇—颜卡礼。

*每日9点整开始抢兑，客户兑换时可任选一种颜色，活动期间每位客户限兑一次。权益一经兑换，非商品质量问题不接受退换。客户自行在兑换前通过礼品页面了解清楚礼品详情。兑换结果以实际兑换成功为准。兑换时，客户需确保卡片至少有9个积分，且使用符合颜卡新户礼活动的卡片才可兑换。

礼品领取注意事项：

*发货信息：礼品收货信息请仔细填写核对，兑换后不接受更改收货信息。中信银行信用卡中心将按填写的收货信息发货，发货周期为成功兑换后两个自然月内。中信银行信用卡中心不承担因收货信息填写错误及手机号无人接听等原因导致的收货问题产生的任何责任。如因节假日期间快递停运，期间兑换的物流类商品将在节后快递上班后陆续发出。如遇快递不支持配送的地区，有权协调客户更改快递地址。

*礼品一经兑换，非商品质量问题不接受退换。权益过期未兑换不再补兑。未成功收到礼品的持卡人最晚可于当期活动结束后1个月内致电中信银行信用卡客服提出异议，未在此期间来电提出异议的，视同持卡人主动放弃所获礼品。

*在法律法规允许的范围内，中信银行信用卡中心保留本次活动最终解释权。
